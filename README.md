`OAuth2::Discord` is a Perl 6 module for authenticating against Discord's OAuth2 API.

## Installation

### ... from zef

```
zef install OAuth2::Discord
```

### ... from source

```
git clone https://github.com/shuppet/p6-oauth2-discord
cd p6-oauth2-discord/ && zef install ${PWD}
```
